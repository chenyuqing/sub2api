# 五、创建 sub2api 数据库

## 1. 创建数据库用户

先进入 PostgreSQL：

```bash
psql postgres
```

进入后执行下面 SQL。把 `换成你的强密码` 改成你自己的密码。

CREATE USER sub2api WITH PASSWORD '换成你的强密码';
CREATE DATABASE sub2api OWNER sub2api;
GRANT ALL PRIVILEGES ON DATABASE sub2api TO sub2api;
\q

psql "postgres://sub2api:换成你的强密码@127.0.0.1:5432/sub2api?sslmode=disable"

# 十四、内存优化建议

8GB Mac 裸机部署后，建议你控制 PostgreSQL 和 Redis 内存。

## 1. Redis 限制内存

打开 Redis 配置：

```bash
brew info redis
```

它会告诉你配置路径，常见是：

```text
/opt/homebrew/etc/redis.conf
```

编辑：

```bash
nano /opt/homebrew/etc/redis.conf
```

找到或添加：

```conf
maxmemory 128mb
maxmemory-policy allkeys-lru
```

重启：

```bash
brew services restart redis
```

## 2. PostgreSQL 不要调太大

Homebrew 默认配置通常已经比较保守。小规模内网使用先不用动。

如果后面你发现内存紧张，再去调：

```text
shared_buffers
work_mem
maintenance_work_mem
```
